推导表达式

推导式又称解析器，使用它可以帮助我们快速简单 地对一个数据容器中的所有元素进行筛选或其它操作。 下面我将用列表推导式来进行简单的举例。

假设我们已经定义了一个元素均为数字的列表：list_a

使用推导式输出一个list_b，其中的元素是列表a中元素的平方：


                list_b = [x**2 for x in list_a]

使用推导式输出一个list_c，其中的元素是列表a中 所有大于十的元素：


                list_c = [x for x in list_a if x > 10]

条件语句和对元素的操作可以同时使用，例如，以 下操作是输出一个list_bc，这个列表中的元素是列表a中所有大于10的元素的平方：


                list_bc = [x**2 for x in list_a if x > 10]

推导表达式和for循环一样，也可以嵌套使用，例如我们可以输出一个list_d，其中的元素均为在一定范围内的点坐标：


                list_d = [(x, y) for x in range(5, 10) for y in range(2, 10)]

其他类型的推到表达式与列表推导式类似，不过元 组推导表达式是把方 括号改为圆括号，字典和集合是把方括号改为花括号，而且字典是对整个键值对进行操作，其它没有什么区别。
